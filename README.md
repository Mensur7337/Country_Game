# 🌍 Country Game - Flag Quiz Adventure

Country Game is a fun web app to test your geography knowledge.  
You see a flag and try to guess the country name, capital, and population.

This project uses the REST Countries API to get real data.

---

## ✨ Features

- 👤 **User System**  
  You can create an account and login.  
  Passwords are saved safely (hashed).

- 🎮 **Difficulty Levels**  
  You can choose:
  - Easy  
  - Medium  
  - Hard  
  Countries are selected by population.

- ⏱️ **Timer**  
  You have 30 seconds for each question.  
  Answer fast to get more points.

- 🏆 **Scoring System**  
  You get points for:
  - Country name  
  - Capital  
  - Population (±10%)

- 🌐 **Leaderboard**  
  - See your last 10 games  
  - Compare with Top 10 players  

- 🎨 **UI Design**  
  - Progress bar  
  - Glassmorphism design  
  - Show / Hide password  

---

## 🛠️ Technologies

- HTML5 & CSS3  
- Vanilla JavaScript  
- REST Countries API  
- Local Storage  

---

## 🎮 How to Play

1. Register or login  
2. Choose difficulty level  
3. Click **Start**  
4. Look at the flag  
5. Write your answers  
6. Click **Answer**  
7. Click **Next Question**
